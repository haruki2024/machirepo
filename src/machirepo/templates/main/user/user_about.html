まちレポについて

「まちレポ」とは？

「まちレポ」は、市民のみなさんがまちの中で見つけたインフラの不具合（道路の陥没、公園の遊具の破損、街灯の故障など）を、スマートフォンから簡単に報告できるアプリケーションです。

撮影・投稿

確認・受付

対応・改善

わたしたちのミッション

このアプリは、市民と行政をつなぐ架け橋となることを目指しています。

みなさんからの「声（レポート）」を集約し、共有することで、まちの問題を早期に発見し、より安全で快適な生活環境を維持することに貢献します。

使い方

  1. ホーム画面の「新規投稿」ボタンを押します。
  2. 「写真」「場所」「カテゴリ」「詳細」を入力します。
  3. 「投稿する」ボタンで内容を確認し、送信します。
  4. 投稿の状況は「投稿履歴の確認」からいつでもチェックできます。
  5. ほかのユーザーの投稿もトップページから閲覧可能です。
